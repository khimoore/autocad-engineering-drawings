# AutoCAD Engineering Drawings

## Overview
This repository contains engineering drawings and layouts created using AutoCAD with X-Ref functionality. The work demonstrates proficiency in drafting standards, layout coordination, and clear technical documentation for engineering applications.

## Tools & Software
- AutoCAD (X-Ref)
- Engineering design and drafting standards
- PDF documentation and drawing exports

## Key Skills Demonstrated
- 2D engineering drafting and layout creation
- External reference (X-Ref) management
- Assembly and layout coordination
- Dimensional accuracy and annotation
- Technical documentation and organization

## Project Details
The drawings included in this repository were developed as part of academic engineering coursework and projects. Files include layouts, assemblies, and documentation created from engineering specifications and rough sketches.

## File Structure
- /drawings – AutoCAD drawing files and layouts  
- /exports – PDF versions of drawings for viewing  
- /documentation – Supporting notes and descriptions  

## Notes
All work shown is for educational and portfolio purposes only.
